# GrowthTrace V1 验收清单

## 1. 验收目标

V1 的目标不是做重型平台，而是交付一个可运行、可演示、可答辩的最小闭环系统，覆盖：

- 注册 / 登录
- 自然语言建档与画像确认
- 成长目标与要求维护
- 成长随记与抽取确认
- 执行任务与打卡
- 阶段诊断与复盘
- Dashboard 总览展示

## 2. 验收范围

### 2.1 在范围内

- 前后端项目可启动
- MySQL 库表可初始化
- 核心页面与接口闭环可访问
- Demo 数据可支持答辩演示
- 关键构建命令可通过

### 2.2 不在范围内

- 多租户
- 复杂权限体系
- 分布式部署
- 多模型编排
- 高并发性能压测

## 5. 运行前置条件

- `DB_USER`
- `DB_PASSWORD`
- `JWT_SECRET`

- `AI_BASE_URL`
- `AI_API_KEY`
- `AI_MODEL`
